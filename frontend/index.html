Welcome to the Docker Project

This is a simple web application demonstrating the use of Docker for deploying a frontend and backend.

About

This project showcases how to use Docker to containerize a web application with a frontend and backend. The frontend is built using HTML, CSS, and JavaScript, while the backend is powered by Node.js.

Features

  • Simple and clean user interface
  • Responsive design
  • Easy deployment using Docker

Getting Started

To get started with this project, ensure you have Docker installed on your machine. You can then build and run the application using Docker Compose.

docker-compose up --build
